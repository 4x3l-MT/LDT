Android Honeycomb (Android 3.0, API nivel 11), lanzado en febrero de 2011, fue la primera versión de Android diseñada exclusivamente para tablets, abandonando momentáneamente los teléfonos para ofrecer una experiencia optimizada en pantallas grandes. Introdujo una interfaz holográfica con elementos como la barra de acción, botones virtuales en pantalla y el sistema de fragmentos, permitiendo a las aplicaciones adaptarse a diferentes tamaños de pantalla de forma modular. Entre sus características destacaron el soporte para procesadores multinúcleo, aceleración gráfica por hardware, navegación mejorada con pestañas, y una experiencia multimedia superior con soporte para video llamadas y códecs avanzados. Honeycomb allanó el camino para la futura unificación de smartphones y tablets que llegaría con Ice Cream Sandwich.
